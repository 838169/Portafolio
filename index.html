Mis proyectos :

Incubadora Casera (Arduino)

Como proyecto final para la culminación de mi técnico en sistemas en el colegio, desarrollé una incubadora casera utilizando Arduino. Este proyecto buscaba automatizar el proceso de incubación de huevos, permitiendo controlar la temperatura y humedad de manera precisa, asegurando un ambiente óptimo para el desarrollo de los embriones.

Virus Detecter (Construct2)

Creé un videojuego 2D llamado Virus Detecter utilizando Construct2. Este juego fue diseñado con mecánicas sencillas y adictivas para combatir virus virtuales en diferentes niveles. El proyecto fue vendido a un grupo de estudiantes que lo utilizaron como parte de sus actividades educativas.

Página Web para Empresa de Huevos (En proceso)

Actualmente estoy trabajando en una página web destinada a una empresa de producción de huevos. Esta plataforma tiene como objetivo mejorar el manejo de inventarios y facilitar el control diario de la producción, optimizando el orden y la eficiencia en los registros de la empresa.

Autoguardado de Compras de Licores

Mi cuarto proyecto consiste en una página web de licores desarrollada con HTML, CSS y PHP. Esta plataforma permite a los usuarios comprar licores disponibles, y una característica innovadora es que el historial del carrito de compras se guarda automáticamente. Así, incluso si se cierra la sesión o se apaga el dispositivo, los usuarios pueden retomar su experiencia de compra sin perder información.
